## Profile 模块 Tab 重构

- 重构 `profile_tab.dart`：Tab 标签从 `['DEMO', 'M3-UI组件', '动画', '歌单']` 改为 `['颜色系统', '卡片组件', '列表组件', '对话框', '网络请求', '其他组件']`
- 内容改为内联展示（非 Get.to 跳转），每个 Tab 对应一个 Section Body Widget，用 SizedBox(height:520) 包裹以提供滚动约束
- 删除了不再需要的 `_SectionItem` 和 `_SectionCard` 类
- 新增可复用 Body 组件：
  - `DialogsSectionBody` — 从 `DialogsSection` 提取 Column 内容
  - `OtherSectionBody` — 从 `OtherSection` 提取 Column 内容
  - `RequestDemoBody` — 从 `RequestDemoPage` 剥离 Scaffold/AppBar，固定高度 520
- `RequestDemoPage` 改为 StatelessWidget，包裹 `RequestDemoBody`（保持向后兼容）
- 清理了 profile_tab.dart 中不再需要的 `design_system_page.dart` 导入
