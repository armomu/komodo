- `MusicPlayerController`：移除所有语音播放相关代码，恢复干净
- `chat_page.dart`：改用 `ChatVoiceController`，`initState` 中 `Get.put(ChatVoiceController())`
- `pubspec.yaml`：新增 `audioplayers: ^6.1.0`

## 聊天录音波形改造（2026-05-08）

**问题**：之前用 `Random()` 随机生成波形条高度，效果假。

**新方案**：`record` 包的 `onAmplitudeChanged(Duration)` → `Stream<Amplitude>`，实时获取麦克风 dBFS 值（-160~0），映射到 0.0~1.0，驱动 20 条滚动波形条。

**改动**：
- `chat_page.dart`：
  - 移除 `dart:math`（不再需要 `Random()`）
  - `_waveHeights` 初始值从 `0.3` 改为 `0.0`
  - 移除 `_waveTimer`，新增 `StreamSubscription<Amplitude>? _amplitudeSub`
  - `_startWaveAnimation()`：订阅 `onAmplitudeChanged(100ms)`，回调中调用 `_dbfsToNormalized()` 更新 `_waveHeights`（左移队列追加新值）
  - `_stopWaveAnimation()`：取消订阅，重置所有高度为 0.0
  - `_dbfsToNormalized(dbfs)`：dBFS -60 以下视为静音返回 0.05，-60~0 映射到 0.05~1.0
- `_RecordOverlayRecording` 中的 `AnimatedContainer` 渲染逻辑不变，数据源已换成真实幅度
- `_amplitudeSub` 在 `dispose()` 中正确 cancel

**架构对比**：

| | MusicPlayerController | ChatVoiceController |
|--|--|--|
| 包 | just_audio + just_audio_background | audioplayers |
| 功能 | 音乐播放 + 系统通知栏 | 聊天语音播放 |
| 生命周期 | `Get.put(permanent: true)` 全局常驻 | `Get.put()` 页面级别 |
