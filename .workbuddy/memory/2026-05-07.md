# 2026-05-07 日志

## komodo 消息模块修复

### 语音消息播放修复
- **问题**：网络语音 URL 无法播放（`setFilePath` 只能用于本地文件）
- **修复**：`music_player_controller.dart` 的 `playVoice` 方法增加网络 URL 判断
  - 网络 URL：用 `setUrl` 播放
  - 本地文件：用 `setFilePath` 播放

### 录音存储路径修复
- **问题**：录音保存在系统临时目录，可能被清理导致录音后无法播放
- **修复**：`chat_page.dart` 改用 `getApplicationDocumentsDirectory()/voice_messages/` 存储
  - 路径：`appDir/voice_messages/voice_timestamp.m4a`

### 图片查看功能
- 新增依赖：`photo_view: ^0.15.0`（缩放）、`share_plus: ^10.0.0`（分享）
- 新建 `lib/pages/message/image_viewer_page.dart`：
  - 全屏黑色背景
  - PhotoView 支持双指缩放（最大 3x）
  - 顶部关闭和分享按钮
  - 底部操作提示文字
- `chat_page.dart` 的 `_buildImageMessage` 的 `onTap` 改为打开图片查看器

## komodo 音乐播放器

### 新增音频波形组件
- 新建 `lib/pages/music/audio_waveform.dart`
- `AudioWaveform`: 32 条波形，根据播放状态动态跳动，颜色从青色渐变到紫色
- `SimpleAudioWaveform`: 简化版，5 条波形
- 已集成到 `music_player_page.dart`，位于黑胶唱片动画下方
- 注意：`audio_visualizer` 包 1.x 版本不支持 just_audio，无法获取真实频率数据，当前使用模拟波形动画

### 新增 Mini 歌词展示行
- 在 `music_player_page.dart` 的歌曲 Tab 中，黑胶唱片下方新增 mini 歌词展示
- 3行歌词，当前播放行居中高亮（白色加粗14px），上下两行淡灰（12px）
- 使用 `Obx` 响应 `currentLyricIndex` 变化自动滚动，带 300ms 动画过渡
- 无歌词时隐藏（`SizedBox.shrink()`）
- 顺手清理了未使用的 `audio_waveform.dart` import

### 技术说明
- Android AudioVisualizer API 只能监听系统媒体应用，无法直接监听第三方播放器
- 如需真实频率可视化，需要原生平台通道或音频解码库（如 ffmpeg）
