## 响应式 DI Demo 拆分与集成

将 `lib/plugins/rx_test.dart` 中的类 GetX 响应式系统拆分为模块化文件并集成到项目中：

### 文件结构
- `lib/plugins/reactive/rx.dart` — Rx<T> / RxList / RxMap 响应式包装
- `lib/plugins/reactive/rx_controller.dart` — RxController 基类
- `lib/plugins/reactive/reactive_injector.dart` — ReactiveInjector 依赖注入容器 + 全局单例 reactiveInjector
- `lib/plugins/reactive/rx_widgets.dart` — RxBuilder / RxConsumer / GetView / RxPage Widget
- `lib/plugins/reactive/reactive_plugin.dart` — barrel export 文件
- `lib/pages/reactive_demo/controllers/counter_controller.dart` — Demo 计数器控制器
- `lib/pages/reactive_demo/services/logger_service.dart` — Demo 日志服务（永久单例）
- `lib/pages/reactive_demo/reactive_demo_page.dart` — Demo UI 页面

### 路由 & 入口
- 路由：`Routes.reactiveDemo = '/reactive-demo'` 注册于 app_routes.dart
- 入口：`buildDemoGrid` 中新增「响应式 DI」条目（橙色电闪图标）
- 访问路径：首页 → 我的 Tab → DEMO → 响应式 DI
