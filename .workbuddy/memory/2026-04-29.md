## 新增音乐播放器页面（仿QQ音乐UI）

### 创建的文件
- `lib/pages/music/music_player_page.dart`：仿QQ音乐深蓝沉浸风格播放器页面

### 功能特点
- 深蓝渐变背景，随当前曲目 accentColor 动态变化
- 顶部 Tab 导航：推荐 / 歌曲 / 歌词，三页内容切换
- 3D黑胶唱机主视觉区：旋转黑胶（AnimationController）+ 唱针臂 + 频谱波形
- 歌曲信息区：歌名/歌手/标签/版权信息 + 点赞按钮
- 功能按钮栏：识曲/均衡/下载/评论/更多（带badge）
- 进度条：SliderTheme自定义样式，时间动态计算
- 播放控制区：循环/上一首/暂停/下一首/播放列表
- 榜单列表：10首曲目可点击切换，当前播放高亮

### 路由注册
- `Routes.musicPlayer = '/music-player'` 加入 `app_routes.dart`
- 转场方式：`Transition.downToUp`（从下弹出，符合QQ音乐交互习惯）

### 首页接入
- `music_tab.dart` 的 `_buildMusicRankingCard` 添加 `GestureDetector`
- 标题行（含 `>` 箭头）和整个卡片均可点击 → `Get.toNamed(Routes.musicPlayer)`

## 播放器功能升级（完整实现）

### 依赖更新
- `pubspec.yaml` 添加 `just_audio: ^0.9.40` 音频播放依赖
- 配置 `sounds/` 文件夹为 asset

### 播放列表
- 从 `sounds/` 文件夹加载两首歌：
  - Manta - 刘柏辛Lexie (AAC格式)
  - 离家出走 - 卫兰 (MP3格式)
- 每首歌配有对应的 LRC 歌词文件

### LRC 歌词解析
- 实现 `_parseLrc()` 方法解析歌词时间戳
- 支持 `[mm:ss.xx]` 和 `[mm:ss.xxx]` 两种格式
- 歌词按时间排序，支持点击跳转

### 歌词同步滚动
- 实时监听播放位置，计算当前歌词行
- 当前歌词高亮显示（放大+白色）
- 自动滚动到当前歌词行（居中显示）
- 非当前歌词灰色显示

### 播放控制
- 播放/暂停/上一首/下一首
- 支持拖动进度条跳转
- 播放完成自动下一首
- 加载状态显示

### Tab 左右滑动切换
- 使用 `PageView` + `TabController` 实现联动
- 左右滑动切换「歌曲」/「歌词」页面
- Tab 指示器跟随滑动动画

### UI 优化
- 顶部显示当前播放歌曲信息
- 底部播放列表横向滚动条
- 精简功能按钮（无边框底色）
